# MindrianOS Plugin for Claude Code

Your AI innovation co-founder. Larry thinks with you through PWS methodology, builds your Data Room as you explore, and chains frameworks intelligently.

**One command to install. Larry starts talking.**

## Quick Start

```
claude plugin install mindrian-os@mindrian-marketplace
```

Then just talk:

> "I have an idea for a platform that connects..."

Larry responds. Your Data Room starts building. No setup required.

Or run a command:

```
/mindrian-os:new-project
```

## What You Get

**Larry** -- a thinking partner modeled on 30+ years of teaching innovation at Johns Hopkins. Not a chatbot. Not a framework dispenser. A provocative, warm, demanding co-founder who thinks WITH you.

**Data Room** -- 8 structured sections that build automatically as you work:
problem-definition, market-analysis, solution-design, business-model, competitive-analysis, team-execution, legal-ip, financial-model

**38 Commands** across 4 categories:

## Optional: Connect Brain

For deeper intelligence powered by a 21K-node knowledge graph:

```
/mindrian-os:setup brain
```

This adds Neo4j + Pinecone integration. Larry gets smarter -- framework chain recommendations, calibrated grading, cross-domain patterns. Everything still works without it.

## Visual Dashboard

```
/mindrian-os:room view
```

Opens a localhost De Stijl dashboard showing your Data Room as an interactive knowledge graph. Zoom, pan, click nodes. Chat box for natural language queries.

## PDF Export

```
/mindrian-os:export thesis    # Investment thesis
/mindrian-os:export summary   # Executive summary
/mindrian-os:export report    # Due diligence report
/mindrian-os:export profile   # PWS profile
```

Professional De Stijl formatted PDFs generated from your Data Room.

## Architecture

```
Plugin (this repo)     -- Skills, commands, agents, hooks
Brain (optional)       -- Neo4j 21K nodes + Pinecone 1.4K embeddings
Room (your workspace)  -- Data Room sections, artifacts, state
```

Everything runs locally. Your data stays on your machine. Brain is optional and remote.
