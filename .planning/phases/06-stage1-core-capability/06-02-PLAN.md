<task type="auto">
  <name>Task 1: Create transcribe-audio script</name>
  <files>scripts/transcribe-audio</files>
  <action>
Create `scripts/transcribe-audio` as an executable Bash script following the pattern from the research doc's code example. Structure:

**Header:**
```bash
#!/usr/bin/env bash
# transcribe-audio: Convert audio file to timestamped speaker-labeled transcript
# Requires: openai-whisper or whisperx, ffmpeg, pyannote.audio
# Run /mindrian-os:setup transcription to install dependencies
# Usage: scripts/transcribe-audio <audio-file> [output-file]
```

**Dependency checks (fail fast with helpful messages):**
1. Check `$1` is provided and file exists.
2. Check `ffmpeg` is installed -- if not: "ERROR: ffmpeg not installed. Run: sudo apt install ffmpeg (Linux) or brew install ffmpeg (macOS)"
3. Check for whisperx first (preferred), then fall back to openai-whisper.
4. If neither: "ERROR: Neither whisperx nor openai-whisper installed. Run: /mindrian-os:setup transcription"

**Two execution paths:**

Path A (whisperx available -- full pipeline):
- Load audio with whisperx.load_audio
- Transcribe with whisperx.load_model('base', device) -- use 'cuda' if torch.cuda.is_available() else 'cpu'
- Align timestamps with whisperx.load_align_model
- If HF_TOKEN environment variable is set: run pyannote diarization via whisperx.DiarizationPipeline, then whisperx.assign_word_speakers
- If HF_TOKEN not set: skip diarization, use "Unknown" for speaker labels, print warning to stderr: "WARNING: No HF_TOKEN set. Transcribing without speaker labels. Run /mindrian-os:setup transcription for full diarization."
- Output format (to stdout or $2):
  ```
  [Speaker_01] 0.0-15.2
  Text of what they said in this segment...

  [Speaker_02] 15.5-28.1
  Their response text...
  ```

Path B (openai-whisper only -- no diarization):
- Load model with whisper.load_model('base')
- Transcribe with model.transcribe(audio_file)
- Output format (no speaker labels):
  ```
  [Unknown] 0.0-15.2
  Text of segment...
  ```
- Print warning: "WARNING: Using openai-whisper without diarization. Install whisperx + pyannote for speaker labels."

**Make executable:** chmod +x

**Model size note in comments:** Default is 'base' for speed. Users can set WHISPER_MODEL env var to 'small', 'medium', or 'large-v3' for better accuracy at the cost of speed and memory.
