- "Themes" -> triggers "convergence themes"

**Chat Query Matching (JavaScript):**
Implement `handleChatQuery(query, intelligence)` function using the pattern from the research doc:

1. Gap queries (pattern: /gap|missing|empty|need/i):
   - Format each gap: "[CONFIDENCE] [section]: [message]"
   - If no gaps: "No gaps detected -- your room coverage is strong."

2. Contradiction queries (pattern: /contradict|conflict|mismatch|inconsisten/i):
   - Format each contradiction: "[section_a] vs [section_b]: [message] (CONFIDENCE)"
   - If none: "No contradictions found between sections."

3. Convergence queries (pattern: /converg|theme|pattern|common|shared/i):
   - Format each theme: "[term] appears across [count] sections"
   - If none: "No convergence patterns yet -- add more artifacts to build connections."

4. Summary queries (pattern: /summar|overview|status|how.*(doing|look|room)/i):
   - Format: "Room: [total] artifacts across [active] sections ([empty] empty). Stage: [venture_stage]."
   - Then list active sections with counts

5. Section-specific queries: match section name words in query, show artifacts + status for that section

6. Edge/pipeline queries (pattern: /feeds|chain|pipeline|flow/i):
   - Show pipeline connections from the graph edges

7. Fallback:
   - "I can answer about gaps, contradictions, convergence themes, and section status. For deeper AI analysis, use /mindrian-os:room in your Claude session."

**Response formatting:**
- Use HTML in responses (not markdown) -- `<br>` for line breaks, `<strong>` for emphasis
- Color confidence levels: HIGH in #A63D2F (red), MEDIUM in #C8A43C (yellow)
- Prefix Larry-style greeting for first response: "Here's what I see in your room..."

**Wire to intelligence data:**
- Access `window.__roomIntelligence` (set by the graph loading code from Plan 01)
- If intelligence is null/undefined, show: "Loading room intelligence... Try refreshing."
