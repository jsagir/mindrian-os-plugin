## Decisions Made

- Fonts sourced from fontsource CDN (Inter) and dharmatype GitHub (Bebas Neue) for reliable TTF downloads
- Profile template filters entries by `metadata.source` field to map methodology outputs to grid cells
- Due diligence uses Jinja2 `namespace(counter=0)` pattern for section numbering across conditional blocks
- CSS @font-face uses `url()` relative paths (not `local()`) for WeasyPrint compatibility with base_url
