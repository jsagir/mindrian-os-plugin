       - h1/h2/h3: Bebas Neue, letter-spacing 0.05em
       - * { border-radius: 0 } (De Stijl: zero radius everywhere)
       - .room-section: break-before: page (explicit page breaks between sections)
       - .accent-bar: 4px solid left border using var(--accent), padding-left 16px
       - .entry: margin-bottom 24px, break-inside: avoid
       - orphans: 3, widows: 3 on paragraphs
       - .doc-title: string-set: doctitle content() (for running header updates)
       - Two-column layout class (.two-col) for executive summary
       - .profile-grid: CSS grid for profile single-page layout

       ```html
         <style>{% block extra_css %}{% endblock %}</style>
       </head>
       <body>

           {% if metadata.venture_name %}
           <p class="venture-name">{{ metadata.venture_name }}</p>
           {% endif %}
           <p class="generated-date">{{ metadata.generated }}</p>
         </div>
         {% block content %}{% endblock %}
       </body>
       </html>
       ```
       Cover page has break-after: page to separate from content.
