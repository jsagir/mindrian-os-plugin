**Helper functions:**
- `stageSymbol(stageName)` — maps stage string (e.g., "discovery", "pre-opportunity") to symbol character
- `edgeSymbol(edgeType)` — maps edge type string to symbol character
- `healthSymbol(entryCount)` — returns ■ (>0), □ (0), or ▪ (partial — when explicitly flagged)
- `colorize(text, colorName)` — wraps text in ANSI color + reset
- `formatEdge(fromSection, toSection, edgeType)` — returns "fromSection →INFORMS→ toSection" with colored edge symbol
- `formatSectionHeader(sectionName, entryCount, stage)` — returns "◎ problem-definition ■ 3 artifacts"
