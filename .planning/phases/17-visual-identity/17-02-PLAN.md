**`renderRoomDiagram(sections)`** вҖ” Takes array of `{ name, entryCount, stage, edges: [{ to, type }] }` objects. Returns a multi-line string showing the room as Unicode box diagram.

Layout algorithm (Claude's discretion on exact sizing):
- Each section is a box: `в”Ңв”Җsection-nameв”Җв”Җв”Җв”Җв”Җв”Җв”җ / в”Ӯ в–  N artifacts      в”Ӯ / в”Ӯ в—Һ Stage           в”Ӯ / в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ`
- Boxes arranged in 2-column grid (adaptive to terminal width, default 80 chars)
- Edges shown between boxes: `вҶ“ INFORMS`, `вҠ— CONTRADICTS`, etc.
- Empty sections (0 entries) show: `в”Ӯ в–Ў EMPTY вҖ” GAP      в”Ӯ`

  - Section names in cream, edge types in semantic colors, GAP in red
