function output(result, raw, rawValue) {
  if (raw && rawValue !== undefined) {
    process.stdout.write(String(rawValue));
  } else {
    const json = JSON.stringify(result, null, 2);
    if (json.length > 50000) {
      const tmpPath = path.join(require('os').tmpdir(), `mindrian-${Date.now()}.json`);
      fs.writeFileSync(tmpPath, json, 'utf-8');
      process.stdout.write('@file:' + tmpPath);
    } else {
      process.stdout.write(json);
    }
  }
  process.exit(0);
}

  <action>
Create the Node.js shared core following the GSD gsd-tools.cjs pattern exactly. No npm dependencies — pure Node.js built-ins only.

**lib/core/index.cjs** — Shared helpers:
- `output(result, raw, rawValue)` — JSON to stdout, large payloads (>50KB) to tmpfile with `@file:` prefix. Replicate GSD pattern exactly.
- `error(msg)` — Write to stderr, exit 1.
- `safeReadFile(filePath)` — Returns file contents or null if not found.
- `PLUGIN_ROOT` — Resolved via `path.resolve(__dirname, '../..')`.

**lib/core/section-registry.cjs** — Section metadata + dynamic discovery:
- `CORE_SECTIONS` object — 8 DD-aligned sections with `label` and `color` properties (from build-graph SECTION_COLORS/SECTION_LABELS).
- `EXTENDED_SECTION_META` object — Pre-assigned metadata for known extensions: opportunity-bank, funding, personas.
- `discoverSections(roomDir)` — Scans `roomDir/*/` directories. Returns `{ core, extended, all, getMeta(name) }`. Excludes: hidden dirs (`.`-prefixed), structural dirs (`meetings`, `team`). A directory qualifies as a section if it contains at least one .md file or has a STATE.md.
- `STRUCTURAL_DIRS` constant — `['meetings', 'team']` (directories that are not sections).

**lib/core/room-ops.cjs** — Room operation wrappers:
- `listSections(roomDir)` — Calls `discoverSections()`, returns `{ sections: [{name, type, label, color}], core_count, extended_count }`.
- `analyzeRoom(roomDir)` — Calls `bash scripts/analyze-room` via `execSync`, returns raw stdout as string. Parse into structured object later (not in Phase 10 — keep it simple, wrap don't rewrite).
- Use `path.resolve(__dirname, '../../scripts')` for script paths. Accept `roomDir` as explicit parameter (never assume CWD).

**lib/core/state-ops.cjs** — State operation wrappers:
- `computeState(roomDir)` — Calls `bash scripts/compute-state` via `execSync`, returns raw stdout as string.
- `getState(roomDir)` — Reads `roomDir/STATE.md`, returns contents or null.
- Use 10-second timeout on execSync calls.

**bin/mindrian-tools.cjs** — Entry point (mark executable with `chmod +x`):
- Shebang: `#!/usr/bin/env node`
- Require all core modules.
- Parse `process.argv.slice(2)` — command is `args[0]`, subcommand is `args[1]`.
- `--raw` flag support: if last arg is `--raw`, pass `raw=true` to output().
- Switch-case routing:
  - `room list-sections [roomDir]` -> `roomOps.listSections(roomDir || './room')`
  - `room analyze [roomDir]` -> `roomOps.analyzeRoom(roomDir || './room')`
  - `state compute [roomDir]` -> `stateOps.computeState(roomDir || './room')`
  - `state get [roomDir]` -> `stateOps.getState(roomDir || './room')`
- Unknown command -> `error('Unknown command: ...')` with usage message.
- Wrap main() in async + catch.

Do NOT add surface-specific logic (no `if (mode === 'cli')` or MCP branches). Core is surface-agnostic. Do NOT rewrite any Bash logic in Node.js — wrap via execSync only.
