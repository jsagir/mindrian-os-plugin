**scripts/analyze-room** — Replace lines 17-26 (hardcoded SECTIONS array) with:
1. Keep a `CORE_SECTIONS` array with the 8 DD names (for gap messaging and section-specific analysis logic).
2. Add dynamic discovery loop:
   - Scan `$ROOM_DIR/*/` directories.
   - Skip hidden dirs (`.`-prefixed), skip structural dirs (`meetings`, `team`).
   - A directory is a section if it contains at least one .md file or has STATE.md.

4. Section-specific gap messages (e.g., "No problem definition found") remain only for core sections. Extended sections get a generic gap message: "STRUCTURAL_GAP:${section}:LOW:Section exists but has sparse content".

1. Keep `SECTION_COLORS` and `SECTION_LABELS` associative arrays for the 8 core sections (display names and De Stijl colors).
