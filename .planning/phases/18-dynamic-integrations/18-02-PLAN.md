**Update `scripts/session-start`:**

After the existing Brain status context line, add integration detection. Insert after the context bridge section (around line 60+):

```bash
# Detect integrations
INTEG_OUTPUT=$(node "${PLUGIN_ROOT}/bin/mindrian-tools.cjs" detect-integrations 2>/dev/null || echo "{}")
INTEG_COUNT=$(echo "$INTEG_OUTPUT" | python3 -c "import sys,json; d=json.load(sys.stdin); print(sum(1 for v in d.values() if v.get('status')=='connected'))" 2>/dev/null || echo "0")

if [ "$INTEG_COUNT" -gt 0 ]; then
  context="${context}\n\nIntegrations: ${INTEG_COUNT} connected"
fi
```

This adds a single line to the session context showing how many integrations are active. The statusline format already exists from Phase 17 visual system — this adds integration count to the context string that Larry reads.
