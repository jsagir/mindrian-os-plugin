  <action>
Create `lib/core/lazygraph-ops.cjs` following existing codebase conventions (strict mode, JSDoc, module.exports at bottom). The module wraps KuzuDB with these exported functions:

**openGraph(roomDir)** — Opens (or creates) the KuzuDB database at `{roomDir}/.lazygraph/`. Creates directory if missing. Calls initSchema. Returns `{ db, conn }`. Use `path.resolve` on roomDir.

**initSchema(conn)** — Creates all node and relationship tables with `IF NOT EXISTS`. Node tables: `Artifact(id STRING PRIMARY KEY, title STRING, section STRING, methodology STRING, created STRING, content_hash STRING)` and `Section(name STRING PRIMARY KEY, label STRING)`. Relationship tables: `INFORMS(FROM Artifact TO Artifact)`, `CONTRADICTS(FROM Artifact TO Artifact, confidence STRING)`, `CONVERGES(FROM Artifact TO Artifact, term STRING)`, `ENABLES(FROM Artifact TO Artifact)`, `INVALIDATES(FROM Artifact TO Artifact)`, `BELONGS_TO(FROM Artifact TO Section)`.

**indexArtifact(conn, roomDir, filePath)** — Indexes a single .md file: extract id (relative path without .md), title (from `# heading`), section (first path segment after room/), methodology (from frontmatter), content_hash (md5 first 8 chars). MERGE the Artifact node. MERGE the Section node. MERGE BELONGS_TO edge. Scan content for `[[wikilinks]]` and create INFORMS edges to matching artifacts. Use string interpolation with single-quote escaping (per research: validate parameterized queries later, use escaping for v1).

**rebuildGraph(conn, roomDir)** — DETACH DELETE all nodes, then walk all sections via `discoverSections(roomDir)`, index every .md file (skip STATE.md, ROOM.md). Return `{ success: true, artifacts: N, sections: N }`.

**queryGraph(conn, cypher)** — Execute Cypher query via `conn.query()`, call `result.getAll()`, return rows array.

**graphStats(conn)** — Count Artifact nodes, Section nodes, and each edge type. Return `{ nodes: { Artifact: N, Section: N }, edges: { INFORMS: N, CONTRADICTS: N, CONVERGES: N, ENABLES: N, INVALIDATES: N, BELONGS_TO: N }, total: { nodes: N, edges: N } }`.

**closeGraph(db)** — Call `db.close()`.

For CONTRADICTS detection: scan artifact content for terms like "however", "contradicts", "unlike", "disagrees" near `[[wikilinks]]` or section references. For CONVERGES: detect when the same term/concept appears in the current artifact AND in 2+ other sections (use existing analyze-room patterns). ENABLES and INVALIDATES: stub detection (create edges only when explicit markers like `enables:` or `invalidates:` appear in frontmatter — full detection is Tier 2 per research open question #3).

Use `node -e "..."` inline for each test assertion (same pattern as existing test scripts).
