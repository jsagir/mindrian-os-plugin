**loadRoomConfig(roomDir) function (MODEL-04):**
- Defaults object: `{ model_profile: 'quality', model_overrides: {} }` (per D-01 default is quality)
- Try room/.config.json first: `path.join(roomDir, '.config.json')`, read with safeReadFile, JSON.parse, spread over defaults
- If no room config, try global: `path.join(os.homedir(), '.mindrian', 'defaults.json')`, same pattern
- Return defaults if both missing
- Catch JSON parse errors and fall through

**parseVentureStage(roomDir) function (MODEL-02):**
- Call getState(roomDir) to get STATE.md contents
- If null, return null
- Match regex `/(?:Venture\s+)?Stage:\s*(.+)/i` to extract stage string
- Trim and validate against `['Pre-Opportunity', 'Discovery', 'Validation', 'Design', 'Investment']` (case-insensitive match)
- Return matched valid stage string or null

**resolveModel(roomDir, agentType) function (MODEL-06):**
5-step resolution cascade per D-03:
1. Per-agent override: `config.model_overrides?.[agentType]` -- if truthy, return it
2. Venture-stage hint: call parseVentureStage, look up `STAGE_HINTS[stage]?.[agentType]`. If value is null, return 'skip'. If value is a string, return it. If agent not in stage hints, fall through.
3. Profile is 'inherit': if `config.model_profile === 'inherit'`, return 'inherit'
4. Profile lookup: look up `MODEL_PROFILES[agentType]?.[profile]`. If found, return it.
5. Default: return 'sonnet'

**resolveCascadeModel(step) function (MODEL-05):**
- Look up `CASCADE_MODELS[step]`
- Return the value (string or null)
- If step not found, return 'sonnet' as fallback

**formatProfileTable(roomDir) function (MODEL-03 support):**
- Load config via loadRoomConfig
- Parse venture stage via parseVentureStage
- For each agent in MODEL_PROFILES, call resolveModel to get resolved model
- Track the resolution source for display: "(override)", "(stage: {stage})", "(profile)", "(default)"
- Build a padded table string similar to GSD's formatAgentToModelMapAsTable: agent name padded to 22 chars, model + source
- Include CASCADE_MODELS section below the agent table
- Return the complete formatted string

**CLI interface (if run directly):**
When invoked as `node lib/core/model-profiles.cjs <subcommand> [args]`:
- `resolve <roomDir> <agentType>` -- print resolveModel result to stdout
- `cascade <step>` -- print resolveCascadeModel result to stdout
- `table <roomDir>` -- print formatProfileTable result to stdout
- `profile <roomDir>` -- print loadRoomConfig result as JSON to stdout

Use the same `if (require.main === module)` pattern as other lib modules.
